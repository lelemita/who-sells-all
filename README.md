## 목적

- 여러 종류의 책을 사려고 할 때, 전부 취급하는 판매자 찾기
- (todo) 전부를 취급하는 판매자가 없는 경우 대안 제시
- (todo) 책이름으로 isbn 연결해주기
    - 키워드(제목, 작가)로 목록 조회 with pagination

## 준비물

- 환경변수: ttbkey = 발급받은 [알라딘 OpenAPI key](https://blog.aladin.co.kr/openapi)

## 화면 구성

모바일과 데스크톱 환경 모두에 최적화된 단일 세로 흐름 레이아웃을 사용하며, 도서 검색 및 추가 기능은 **풀스크린 모달 팝업**(또는 바텀 시트)으로 분리하여 화면 복잡도를 낮춥니다.

### 1. 메인 화면

- **도서 추가 섹션**
  - `[+ 도서 추가하기]` 버튼 (클릭 시 도서 검색 팝업 오픈)
- **담은 목록 (장바구니)**
  - 전체 선택 체크박스, 선택 항목 일괄 삭제 버튼
  - 목록 구성: 책 이름, 저자, 출판사, 정가를 나타내는 테이블 (각 행 끝에 개별 `[삭제]` 버튼)
- **분석 실행 섹션**
  - `[전부 취급하는 판매자 찾기]` 검색 버튼 (담은 책 중 선택된 항목 기준)
- **판매자 검색 결과** (일괄 최적 정렬 리스트)
  - 정렬 옵션 필터: `[배송비 포함 최저가순 ▾]`
  - 목록 구성: 판매자 이름(지점명), 매치된 도서 수 / 전체 도서 수, 총 상품 가격, 배송비 정보, `[주문하러 가기 ↗]` 버튼
  - 결과가 없을 경우: *"선택하신 책들을 동시에 취급하는 판매자가 없습니다."* 대안 안내 메시지 노출.

### 2. 도서 검색 팝업 (모달)

- **헤더 영역**
  - 도서 검색 입력창 (제목, 저자 등 키워드 검색) 및 `[찾기]` 버튼
  - 우측 상단 `[닫기 X]` 버튼
- **도서 검색 결과 목록** (무한 스크롤 페이징)
  - 목록 구성: 표지 이미지, 책 이름, 저자, 출판사, 출판년도, 정가, `[담기]` 버튼
  - `[담기]` 버튼 클릭 시 메인 화면의 담은 목록에 추가되며, 버튼이 `[담김 ✓]`으로 변경되고 하단 토스트 피드백 노출.
  - 검색 결과가 없는 경우: *"검색 결과가 없습니다. 책 제목을 다시 확인해주세요."* 메시지 노출.

## 주의사항

- totalPrice에 배송비는 반영되어 있지 않음
